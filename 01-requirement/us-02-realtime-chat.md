# US-02: Real-Time Chat with Cool Effects

**As a** participant,
**I want to** send and receive messages in real time with fun visual effects,
**So that** the chat feels engaging and dev-culture-appropriate rather than boring.

## Acceptance Criteria

- [ ] Messages appear in real time for all participants in the session without page refresh
- [ ] Each message displays the sender's generated name and avatar
- [ ] Messages support plain text; code snippets render in a monospace font with syntax highlighting (at minimum triple-backtick blocks)
- [ ] New messages animate in (e.g. slide-in or fade-in effect)
- [ ] A participant can send a message by pressing Enter or clicking a Send button
- [ ] The chat auto-scrolls to the latest message; a "jump to bottom" button appears when the user has scrolled up
- [ ] At least one fun effect exists — e.g. emoji reactions float up the screen, or a confetti burst when someone sends 🎉
- [ ] The participant list (names + avatars) is visible in a sidebar or header showing who is currently in the session
- [ ] Messages from the instructor are visually distinguished (e.g. badge or highlight color)
