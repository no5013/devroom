# US-02: Real-Time Chat with Cool Effects

**As a** participant,
**I want to** send and receive messages in real time with fun visual effects,
**So that** the chat feels engaging and dev-culture-appropriate rather than boring.

## Acceptance Criteria

- [x] Messages appear in real time for all participants in the session without page refresh
- [x] Each message displays the sender's generated name and avatar
- [x] Messages support plain text; code snippets render in a monospace font with syntax highlighting (at minimum triple-backtick blocks)
- [x] New messages animate in (e.g. slide-in or fade-in effect)
- [x] A participant can send a message by pressing Enter or clicking a Send button
- [x] The chat auto-scrolls to the latest message; a "jump to bottom" button appears when the user has scrolled up
- [x] At least one fun effect exists — e.g. emoji reactions float up the screen, or a confetti burst when someone sends 🎉
- [x] The participant list (names + avatars) is visible in a sidebar or header showing who is currently in the session
- [x] Messages from the instructor are visually distinguished (e.g. badge or highlight color)
